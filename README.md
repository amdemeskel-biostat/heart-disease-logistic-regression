# Heart Disease Logistic Regression Analysis

## Overview

This project investigates factors associated with heart disease using multivariable logistic regression and evaluates model performance using ROC analysis.

## Author

Amdemeskel Mulugeta Bekele

MPH Biostatistics Candidate

Data Analyst & Public Health Researcher

## Dataset

Publicly available Heart Disease Dataset obtained from Kaggle.

## Objectives

* Describe participant characteristics
* Identify predictors of heart disease
* Develop a logistic regression model
* Evaluate predictive performance

## Statistical Methods

* Descriptive Statistics
* Chi-square Tests
* Welch t-tests
* Multivariable Logistic Regression
* ROC Analysis
* Variance Inflation Factors (VIF)

## Key Results

* Accuracy: 87.0%
* Sensitivity: 83.2%
* Specificity: 90.2%
* AUC: 0.933
